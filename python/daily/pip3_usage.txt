파이썬을 사용할 때 온라인 상에 존재하는 모듈들을 import해서 사용해야하는데
보통 없는 경우가 많은거 같다. 

예를 들어 지금 마인크래프트를 파이썬으로 구동하고 싶은데

>>> from mcpi.minecraft import Minecraft

를 해야하는데 실행하였더니 그런 모듈이 존재하지 않는다고 출력되었다.
( 현재는 pip3를 이용하여 install 함 )
 
	$ sudo apt-get install python3-pip
	$ pip3 install mcpi

와 같이 터미널에 입력하면 pip3 를 통하여 모듈을 인스톨 할 수 있음

pip3의 사용법을 더 알고 싶다면 터미널에

	$ pip3 

라고 입력하면 옵션들과 사용법에 대해서 잘 설명해주니 참고하도록하자
