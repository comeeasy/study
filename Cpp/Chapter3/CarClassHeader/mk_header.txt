헤더파일, 소스파일을 이용하여 클래스를 분할 하여 저장해둔다.

헤더파일	: 보통 C++에서는 클래스의 기본 틀을 선언하며, 
		  헤더를 중복 include 하지 않기위해 그에 해당하는 이름의 
		  헤더 파일의 이름을 define하여 선언해둔다
		  ex )// #ifndef __CAR_H__
		      // #define __CAR_H__
		      // .... (소스)
		      // #endif
		  소스 부분에 클래스의 기본 틀을 선언

소스 파일	: 위에서 틀을 만들었다면 이제는 실제 내용물을 담을 파일이다.
		  // #include"Car.h" 
		  와 같이 위 헤더파일을 include 시켜주고, 그 밑에 Car.h 에서 
		  선언했던 함수들을 작성한다.

main 파일	: main함수에는 
		  // #include"Car.h" 만 하면 Car.h 에서 선언한 헤더파일도 
		  포함시키기 때문에 뭘 더 포함하지 않아도 괜찮다.
	  	  이제는 Car.h 를 가지고 필요한 함수를 작성하면 된다.

실행파일 만드는 방법 :

	$ g++ Car.h Car.cpp Main.cpp -o Main

	위와 같이 실행시키면 Car.h, Car.cpp Main.cpp 을 링크파일로 먼저 
	컴파일 한 뒤 저 셋을 모두 링크시켜 Main 실행파일을 만들어라
	라는 프롬프트 인듯 정확히는 모르겠지만 실행은 된다!
