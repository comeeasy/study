셸스크립트를 작성한 파일의 확장자는 .sh 이어여한다.
셸 스크립트를 실행하는 방법은
	1. $ sh file-name.sh
	2. $ chmod +x file-name.sh
	   $ ./fild-name.sh
두 가지의 방법이 있다.

< 변수 >
	1. 셸 스크립트에서는 변수를 사용하기 전에 미리 선언하지 않으며,
		처음 변수에 값이 할당되면 자동으로 변수가 생성된다.
	2. 변수에 넣는 모든 값은 'String'으로 간주된다.
	3. 변수 이름은 대소문자를 구분한다.
	4. 변수를 대입할 때, '=' 좌우에는 공백이 없어야 한다.
	5. 터미널에서도 사용가능하다 ㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋ
	6. alias ls="echo 안병철 멍청이" 라고 하면 ls 명령어 대신 
	   $ echo 안병철 멍청이가 입력된다.
	7. $가 들어간 문자열을 출력하려면 ''로 묶어주거나 \를 붙여야한다.
	8. read myvar 처럼 입력 받은값을 변수에 할당해줄 수도 있다.

< 숫자 계산 >
	변수에 넣은 값은 모두 문자열로 취급한다고 했다. 만약 변수에 있는 값에
	'+', '-', '*', '/' 등의 연산을 하려면 'expr' 키워드를 사용해야한다.  		numcalc.sh에 소스를 적음 
	단, 수식과 함께 꼭 키보드 숫자 1 왼쪽에 잇는 역따옴표 '`'로 묶어야한다
	' 와 ` 는 다르다 ㅋㅋㅋㅋㅋㅋㅋ 그리고 수식에 괄호나 '*'를 사용하려면
	예외적으로 역슬레시 (\)를 붙여줘야한다.
