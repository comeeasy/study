<학습 목표>
	- 데이터 처리 프로세스를 이해한다.
	- 데이터 분석 기법 중 시각화를 이해한다.
	- 데이터 분석 기법 중 공간 분석을 이해한다.
	- 데이터 분석 기법 중 탐색적 자료 분석을 이해한다.

<눈높이 체크>
	- 대기업에서는 데이터 분석을 위해 DW(Data-Warehouse), DM(Data-Mart)에서 데이터를 추출해 온다.
	- 데이터 추출이 가능한 기업내 여러 시스템의 이름과 프로세스를 이해하면 보다 효과적으로 분석 데이터 마트를
	  구성 할 수 있다.
	
	- 시각 분석 : 데이터를 도표나 그림으로 한 눈에 나타내는 가장 낮은 수준의 분석, 그러나 가장 필수적임
	- 공간 분석 : 공간적 차원과 관련된 속성을 지도 위에 시각화하는 방법
	- 탐색적 분석 : 다양한 차원과 값을 조합해 특이한 점이나 의미있는 사실을 도추하는 분석.

----------------------------------------------------------------------------------------------------------------
<1절: 데이터 분석 기법의 이해>
	1. 데이터 처리
		1) 개요
			- 데이터 분석은 통계에 기반하고 있지만 학문보다는 실용적인 의미가 더 강한 분야이다.
		2) 활용
			- 대기업은 DW, DM을 통해 분석 데이터를 가져와서 사용한다.
			- 신규 시스템이나 DW에 포함되지 않은 데이터인 경우, 기존 운영시스템(Legacy)이나, 스테이징 영역
			  (staging area)과 ODS(Operational Data Store)에서 데이터를 가져와 DW에서 가져온 내용과 결합하여 사용함
			-> 그러나 가급적이면 클린징 영역인 ODS에서 데이터를 전처리 후 DW, DM과 결합하여 활용하는 것이 이상적이다.
		3) 최종 데이터 구조로 가공
			[1] 데이터 마이닝 분류
				- 분류값과 입력 변수들을 연관시켜 인구통계, 요약변수, 파생변수 등을 산출한다.
			[2] 정형화된 패턴 처리
				- 비정형 데이터나 소셜 데이터는 정형화한 패턴으로 처리해야한다.
				- 비정형 데이터: DBMS에 저장됐다가 텍스트 마이닝을 거쳐 DM과 통합한다.
				- 관계형 데이터: DBMS에 정장되서 사회 신경망분석을 거쳐 분석결과 통계값이 마트와
								 통합되서 활용된다.

	2. 시각화(시각화 그래프)
		- 가장 쉽지만 가장 필수적이고 가장 효율적일 수 있다.
		- 대용량 데이터를 다루는 빅데이터 분석에 시각화는 필수이다.
		- 탐색적 분석을 할 때 시각화는 필수이다.
		- SNA(Social Network Analysis)을 할 때 자주 활용된다.

	3. 공간 분석(GIS)
		- 공간 분석(Spational Analysis)은 공간적 차원과 관련된 속성들을 시각화 하는 분석이다.
		- 서울 지도를 가져다 놓고 인구수를 시각화하는 표를 생각할 수 있다.

	4. 탐색적 자료 분석(EDA)
		1) 개요
			- 탐색적 분석: 다양한 차원과 값을 조합해가며 특이한 점이나 의미 있는 사실을 도출하고
			  			   분석의 최종 목적을 달성해가는 과정이다. 
			- 데이터의 특징과 내재하는 구조적 관계를 알아내기 위한기법들의 통칭이다.
		2) EDA의 4가지 주제
			[1] 저항성의 강조
			[2] 잔차 계산
			[3] 자료변수의 재표현
			[4] 그래프를 통한 현시성
		3) 탐색적 분석의 효율 예
			- KDD, CRISP-DM에서 언급했던 바와 같이
				데이터 이해 
				변수 생성
				변수 선택
			  의 단계에서 활용되고 있다.

	5. 통계 분석
		1) 통계
			- 어떤 현상을 숫자, 표, 그림의 형태로 나타내는 것
		2) 기술 통계
			- Descriptive Statistics
		3) 추측 통계
			- 표본 통계량으로 모집단의 통계량을 추정

	6. 데이터 마이닝
		1) 개요
			- 대표적인 고급 데이터 분석법으로 
			  대용량의 자료
			  로 부터 정보를 요약하고 미래에 대한 예측을 목표로 자료에 존재하는
			  관계, 패턴, 규칙 등
			  을 탐색하고 이를 모형화함으로써 이전에 알려지지 않은 유용한 지식을 추출하는 법
		2) 방법론
			- DB에서의 지식탐색: DW에서 DM을 만들어가며 각 데이터들의 속성을 사전분석하여 조사
			- Machine Learning : 인공신경망(ANN), 의사결정나무, 클러스터링, 베이지안 분류, SVM등이 있다.
			- Pattern Recognition: 장바구니 분석, 연관 규칙등이 있다.
		3) 활용 분야
			- 데이터 베이스 마케팅
			- 신용평가 및 조기 경보 시스템
			- 생물 정보학
			- 텍스트 마이닝
